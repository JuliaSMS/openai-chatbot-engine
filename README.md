## 🤖 OpenAI Chatbot Engine

Aplicação em Python para interação com modelos de linguagem da OpenAI, com histórico de conversas e sistema de moderação de conteúdo.

### 🔥 Funcionalidades
- Chat com modelo GPT
- Histórico de mensagens
- Sistema de moderação de conteúdo
- Estrutura organizada para expansão

### 🛠️ Tecnologias
- Python
- OpenAI API
- dotenv

### 💡 Objetivo
Demonstrar a base de integração com LLMs, incluindo controle de contexto e segurança de entrada.
